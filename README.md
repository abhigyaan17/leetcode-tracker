# 🚀 LeetCode Tracker + Smart Analyzer

A DSA-focused project that tracks coding problems and intelligently analyzes strengths, weaknesses, and recommends what to solve next.

---

## 📌 Overview

This project is a personal analytics system inspired by platforms like LeetCode. Instead of just tracking solved problems, it helps identify weak areas and suggests targeted practice using core Data Structures & Algorithms.

---

## 🎯 Features

### ✅ Core Features

* 📊 Track solved/unsolved problems
* 🔍 Filter by topic, difficulty, and status
* 📈 View progress analytics (topic-wise & difficulty-wise)
* 🧠 Detect weak areas automatically
* 🎯 Smart recommendation system for next problems

---

### 🔥 Upcoming Features

* 📅 Daily streak tracker
* 📊 Timeline visualization
* 🧩 Topic dependency graph (Graph-based learning path)
* 💾 Persistent storage (JSON/Database)

---

## 🧠 DSA Concepts Used

* HashMaps (Tracking & counting problems)
* Sorting Algorithms (Ranking weak areas)
* Priority Queue / Heaps (Recommendation system)
* Graphs (Topic dependency mapping - planned)
* Searching Algorithms
